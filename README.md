# DEEP LEARNING: ANÁLISE DE SENTIMENTOS

Este projeto apresenta a aplicação de modelos de aprendizado profundo para resolver tarefas de Processamento de Linguagem Natural. 
O objetivo principal é demonstrar como técnicas modernas de NLP podem compreender textos, extrair informações relevantes e identificar sentimentos expressos em linguagem humana.

O projeto aborda conceitos fundamentais de NLP, incluindo a representação de linguagem em formato numérico, preparação de dados textuais, treinamento de modelos e avaliação de desempenho. 
Também inclui a implementação de modelos capazes de classificar sentimentos, responder perguntas e realizar outras tarefas comuns da área.

A proposta mostra como algoritmos avançados, especialmente modelos de Deep Learning, contribuem para aproximar a linguagem humana do processamento computacional
permitindo analisar textos de forma automatizada e eficiente.
